数据类型

基本类型

格式： 原始数据类型 / 大小(bit) / 默认值 / 取值范围

  • byte / 8 / 0 / -128(-2^7) ~ 127(2^7-1)
  • short / 16 / 0 / -32,768(-2^15) ~ 32767(2^15-1)
  • int / 32 / 0 / -2^31 ~ 2^31-1
  • long / 64 / 0L / -2^63 ~ 2^63-1
  • float / 32 / 0.0f / 参考链接
  • double / 64 / 0.0d / 参考链接
  • boolean / unknow / false
  • char / 16 / ‘\u0000’ / ‘\u0000’(0) ~ ‘\uffff’(65,535)

虽然编译器会替未初始化的字段设置一个合理的默认值，但不建议依赖编译器的初始化功能。

局部变量略有不同，编译器不会为其初始化，请尽可能地保证在使用之前为其分配一个值。
